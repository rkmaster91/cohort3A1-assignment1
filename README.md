# Pokémon Landing Page

**Assignment 1**

A Pokémon-inspired landing page built using **HTML** and **CSS** while learning frontend development.

## Preview

![Sorry Sir/Ma'am agar pic load nhi ho payi](./Screenshot.pngpng)

---

## Features

* Pokémon-inspired landing page
* Split-screen layout
* Pure CSS Poké Ball watermark (no image used)
* Smooth hover animation on Pikachu
* Modern UI with rounded corners
* Simple and clean design

---

## Concepts Used

* HTML5
* CSS3
* Flexbox
* Positioning (`relative` & `absolute`)
* Transform
* Transition
* Overflow Hidden
* Border Radius
